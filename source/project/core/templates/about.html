Généralités

Présid-apps est un petit site sans prétention visant à référencer les applications web et mobile traitant de la présidentielle française de 2012.

Il est édité par Stéphane Angel, et hébergé sur dotcloud.com

Contact

Pour toute remarque sur le contenu (auteurs, catégories...) ou pour proposer de nouvelles applications, vous pouvez me contacter via Twitter : @Twidi.
Les petits tweets rigolos seront peut-être lus en priorité, tout dépend de l'âge du capitaine ;)

Technique

Ce site est développé en Django, utilisant html5 boilerplate et twitter bootstrap pour l'interface.

Le code source est disponible sur http://github.com/twidi/fpe2aa.
